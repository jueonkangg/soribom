# 5. 안전 관리

## 원칙

> 실제 보청기·인공와우에 우리가 만든 소리를 직접 보내는 실험은 **하지 않는다.**

## 소리 크기

### 왜 리미터가 필요한가

보청기와 인공와우에는 제조사가 설정한 **최대 출력(MPO)** 보호 장치가 내장되어 있습니다.
그렇다면 우리가 음량을 조절할 필요가 없어 보이지만, 그렇지 않습니다.

**시연·개발 단계의 출력 대상은 일반 블루투스 이어버드·스피커**입니다.
이 장치들에는 보청기 같은 MPO 보호가 **없습니다.** 따라서 우리 소프트웨어 쪽에
AGC와 세이프티 리미터를 두는 것이 맞습니다.

### 조치

| 항목 | 내용 |
|---|---|
| AGC | 입력 음량을 자동으로 알맞은 범위로 조절 |
| 세이프티 리미터 | 출력이 설정 상한을 넘지 못하도록 하드 클리핑 |
| 검증 | 디지털 소음계로 실제 출력 dB를 측정해 기록 |

## 실험 참가자 보호

- 모의 저청력 조건의 소음은 **안전한 크기, 짧은 시간**으로 제한
- 참가자는 언제든 중단할 수 있음을 사전에 고지
- 실험 전 보호자 동의 취득
- 불쾌감을 줄 수 있는 콘텐츠 미사용

## 전기 안전

- 국가 안전 인증을 받은 정품 전원 어댑터만 사용
- 노후 부품·전선 사용 금지
- 배선 작업은 전원을 분리한 상태에서 진행

## 개인정보

- 교실 음성은 기기 밖으로 나가지 않음 (완전 오프라인)
- 실험 데이터는 익명 기록
- 개인정보가 포함된 문서는 이 저장소에 커밋하지 않음
