### 7/21 (화) — 합치기

**오늘이 가장 중요한 날입니다.** 세 조각이 한 화면에서 같이 돌아야 합니다.

- 자막 + 방향 + 입력창을 **한 창**에
- I2S 스피커 배선 (`jetson-io.py` 실행 → 재부팅 → `aplay -l`)
- `config.yaml`의 `tts.output_device` 채우기

> 합치는 건 항상 예상보다 오래 걸립니다. 각자 만든 게 서로 충돌합니다.
> **화요일에 안 되면 수요일까지** 씁니다. 그래서 여유를 뒀습니다.

### 7/22 (수) — 숫자 만들기

기능이 도는 것만으로는 부족합니다. **측정해야 합니다.** (CLAUDE.md 8장)

| 지표 | 어떻게 |
|---|---|
| **WER** | 정해둔 문장 20개를 읽고 틀린 단어를 센다. 조용할 때 / 시끄러울 때 |
| **자막 지연** | 말이 끝나고 자막이 뜰 때까지. 1패스 / 2패스 각각 |
| **DOA 오차** | 바닥에 각도 표시하고 8방향 측정 |
| **TTS 지연** | 엔터 → 소리 시작까지 |

**이미 좋은 자료가 하나 있습니다:**

```
small        → 과막성   ❌
medium 1.5GB → 광학성   ❌
small + 단어 4개 → 광합성 ✅
```

같은 방식으로 문장 20개를 재면 **표 하나가 나옵니다.** 발표의 핵심 슬라이드입니다.
